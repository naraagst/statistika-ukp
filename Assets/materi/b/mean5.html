Rata-Rata Gabungan dari
Dua Buah Data

Misalkan terdapat dua kelompok data berikut.

Kelompok data pertama terdiri atas $n_1$ nilai data yaitu $x_1,x_2,x_3,...,x_{n_1}$ dan memiliki rata-rata $\bar{x_1}$.
Kelompok data kedua terdiri atas $n_2$ nilai data yaitu $x_1,x_2,x_3,...,x_{n_2}$ dan memiliki rata-rata $\bar{x_2}$.

Rata-rata gabungan kedua kelompok tersebut dapat dihitung menggunakan rumus berikut.

dengan:
$\bar{x}_{gab}$ = rata-rata gabungan
$n_1$ = banyak data kelompok 1
$\bar{x}_1$ = rata-rata nilai data kelompok 1
$n_2$ = banyak data kelompok 2
$\bar{x}_2$ = rata-rata nilai data kelompok 2

Silahkan pelajari contoh soal dan jawab pertanyaan di bawah ini.

Contoh Soal Pertanyaan

Soal :
Rata-rata berat badan 13 siswa putra 55 kg, sedangkan rata-rata berat badan 7 siswa putri 48 kg. Hitunglah rata-rata berat badan seluruh siswa.

Pembahasan :

Misalkan:
$\bar{x}_1$ = 55   dan $n_1$ = 13,
$\bar{x}_2$ = 48   dan $n_2$ = 7

Jadi, rata-rata berat badan seluruh siswa adalah 52,55

Pembahasan :

Misalkan :
$\bar{x}_1$ = rata-rata nilai 4 siswa
$x_2$ = nilai seorang siswa yang tidak diikutkan
Maka $\bar{x}_1$ = 60, $n_1$ = 4, dan $n_2$ = 1.
